# 贡献指南

感谢你对「勇哥餐饮智慧」项目的关注！欢迎任何形式的贡献，共同让这个知识库更完善、更有价值。

---

## 可以贡献什么

### 1. 纠错与更新
- 修正知识库中的事实错误
- 补充更新的案例数据
- 完善不准确的分析内容

### 2. 内容补充
- 补充特定品类的深度分析（如奶茶、炸鸡、烤肉等）
- 分享真实的创业案例（请脱敏处理，隐去姓名和具体位置）
- 补充特定城市/商圈的选址经验
- 补充翻身成功的正向案例

### 3. 技术贡献
- 改进 Agent 提示词
- 开发新的部署方式（如接入企业微信、钉钉等）
- 优化知识库结构

### 4. 传播与推广
- 分享给正在考虑做餐饮的朋友
- 翻译为其他语言版本
- 制作科普图文或视频

---

## 贡献规范

### 内容要求
- 所有内容必须为**原创表述**，不直接复制原视频台词
- 引用案例请脱敏处理（隐去真实姓名、具体地址）
- 保持客观、务实的风格，以实际帮助到创业者为目的
- 数据型信息请尽量标注时间（如"2025年案例"）

### 敏感话题
- 涉及勇哥个人的内容，请保持尊重
- 不对任何具体品牌做负面商业评价（可描述现象，不点名批判）
- 如有侵权顾虑，请在 Issue 中说明

### 提交方式
1. Fork 本仓库
2. 创建你的分支（`git checkout -b feature/你的贡献`）
3. 提交更改（`git commit -m '描述你的更改'`）
4. 推送到分支（`git push origin feature/你的贡献`）
5. 发起 Pull Request

---

## 行为准则

- 尊重每一位贡献者
- 以帮助餐饮创业者为首要目的
- 不做商业推广
- 不发布未经验证的信息
